# 无限制呼叫

- 对话命令 开辟了对话F0指令
- 格式是F0 ZZ XX YY
- 仿照F9/F8命令 F9/F8 XX YY
- 实际上是变相映射切页
- 示例如下
    - 首先 记下地址 例如7c010
    - 然后 7c010/2000=3E 余数是10
    - 那么ZZ就是3e 余数应该+8000-10等于新数字
    - 新数字对应XX YY
    - 如果想仿造F8的话 ZZ应该+80等于新ZZ
    - 如果想仿造F9的话 ZZ不变
    - 这次切页命令就应该是F0 3E 00 80（F9版本） F0 BE 00 80
    - 配合新参数临时内存可达到无限切页和任意调取的效果
